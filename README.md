# Ruby Muse
A music library management cli tool.

# Getting Started
1. `git clone https://github.com/andela-akabiru/checkpoint-one-music-library.git`
2. `cd checkpoint-one-music-library.git`
3. `bundle install`
4. `./bin/musiclibrary` 

# Usage
Some basic commands include: 

 - `list songs`
 -  `list artists`
 -  `list genres`
 - `play song`
 - `list artist`
 - `list genre`
